## 创建工程并导入插件


```
拷贝cordova-plugin-ipsmapSDK 内容到项目中,执行以下命令进行生产 IOS 和 Android 平台的插件
cordova plugin add ./cordova-plugin-ipsmapSDK
cordova plugin remove cordova-plugin-ipsmapSDK
如果已经添加过相关插件,请先移除,在进行添加

```
